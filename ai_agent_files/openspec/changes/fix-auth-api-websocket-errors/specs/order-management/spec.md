## ADDED Requirements

### Requirement: Order Geometry Support
Система SHALL корректно обрабатывать геометрию маршрутов заказов с использованием PostGIS и GeoAlchemy2.

#### Scenario: Получение заказа с геометрией маршрута
- **WHEN** запрашивается заказ через API `/api/v1/orders`
- **THEN** система извлекает геометрию маршрута из PostgreSQL (PostGIS)
- **AND** геометрия корректно сериализуется в Pydantic модель с использованием Shapely
- **AND** возвращается полный заказ включая поле `route_geometry` в формате GeoJSON

#### Scenario: Обработка заказа без геометрии
- **WHEN** запрашивается заказ без сохраненной геометрии маршрута
- **THEN** система возвращает заказ с `route_geometry: null`
- **AND** не возникает ошибок при сериализации
