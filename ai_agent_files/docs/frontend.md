# Фронтенд

Фронтенд приложение для диспетчеров и администраторов.

## Стек технологий

- **Vite**: Сборщик проекта (быстрый HMR).
- **React**: UI библиотека.
- **TypeScript**: Типизация кода.
- **Ant Design**: Библиотека UI компонентов (таблицы, формы, кнопки).
- **React Query (@tanstack/react-query)**: Управление серверным состоянием, кэширование и синхронизация данных.
- **Zustand**: Менеджер глобального состояния (клиентское состояние).
- **Leaflet / React-Leaflet**: Отображение карт и маркеров.
- **Vis-Timeline**: Визуализация расписания заказов (Gantt chart).

## Основные модули

### Карта (Live Map)
Использует Leaflet для отображения:
- Позиций водителей в реальном времени (поллинг или WebSocket).
- Маршрутов заказов.

### Таймлайн (Timeline View)
Использует `vis-timeline` для отображения заказов во времени, с возможностью Drag-and-Drop редактирования.

### Управление состоянием (Store)
- Используется **React Query** для получения списков водителей и заказов. Это обеспечивает актуальность данных и упрощает работу с загрузкой/ошибками.
- **Zustand** используется для хранения UI-состояний (выбранный драйвер, фильтры карты, состояние модальных окон).

## Команды

- `npm run dev`: Запуск локального сервера разработки.
- `npm run build`: Сборка продакшн бандла (результат в `dist/`).
- `npm run preview`: Предпросмотр собранного бандла.
