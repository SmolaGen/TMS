# Walkthrough: Phase 6 - Design & UX Improvements

## Обзор изменений
В рамках Фазы 6 были внедрены следующим улучшения дизайна и пользовательского опыта:

1.  **Дизайн-система**:
    *   Создан файл конфигурации темы `src/theme.ts` с поддержкой светлой и темной тем.
    *   Внедрены CSS-переменные в `src/App.css` для стандартизации цветов, отступов и теней.
    *   Обновлены стили компонентов (Timeline, Map) для поддержки темной темы.

2.  **Управление темой**:
    *   Реализован хук `useTheme` для переключения тем и сохранения выбора пользователя.
    *   Добавлен компонент `ThemeToggle` в `HeaderBar` для переключения темы из интерфейса.
    *   Добавлена поддержка системной темы.

3.  **Горячие клавиши**:
    *   Реализован хук `useKeyboardShortcuts`.
    *   Добавлены глобальные шорткаты: `D` (смена темы).
    *   Добавлены шорткаты Dashboard: `N` (новый заказ), `Escape` (закрыть модальное окно).

## Процесс верификации

### 1. Проверка тем
1.  Откройте приложение.
2.  В правом верхнем углу (в шапке) найдите иконку солнца/луны.
3.  Нажмите на нее, чтобы переключить тему.
4.  Убедитесь, что:
    *   Фон приложения меняется (светлый/темный).
    *   Компоненты Ant Design (кнопки, инпуты) меняют цвета.
    *   Карта (Leaflet) и Timeline (visjs) выглядят корректно в темном режиме.
5.  Нажмите клавишу `D` на клавиатуре. Тема должна переключиться.

### 2. Проверка горячих клавиш
1.  Перейдите на Dashboard.
2.  Нажмите клавишу `N`.
3.  Должно открыться модальное окно "Новый заказ".
4.  Нажмите клавишу `Escape`.
5.  Модальное окно должно закрыться.
