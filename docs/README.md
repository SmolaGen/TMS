# Документация проекта TMS (Transport Management System)

Добро пожаловать в документацию проекта TMS. Это система управления транспортом, построенная на микросервисной архитектуре с использованием FastAPI, React и географических сервисов.

## Структура документации

В этой папке вы найдете подробные руководства по всем аспектам системы:

1. [Установка и запуск](./setup.md) - Инструкция по развертыванию локального окружения.
2. [Бэкенд и API](./backend.md) - Архитектура серверной части, описание сервисов и API.
3. [Фронтенд](./frontend.md) - Описание клиентской части приложения.
4. [Развертывание](./deployment.md) - Инструкции по деплою в продакшн (Docker Compose).
5. [Тестирование](./testing.md) - Чек-лист тестов перед сдачей проекта.
6. [Рекомендации по улучшению](./advice.md) - Советы по оптимизации и развитию проекта.

## Краткий обзор стека

- **Backend**: Python 3.11+, FastAPI, PostgreSQL (PostGIS), Redis.
- **Frontend**: TypeScript, React, Vite, Leaflet, Ant Design.
- **Infra**: Docker, Docker Compose, Nginx (предполагается).
