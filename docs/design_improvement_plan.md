# План улучшения дизайна и доведения интерфейса до премиального уровня

Этот план направлен на трансформацию текущего интерфейса TMS в современный, визуально привлекательный и функционально завершенный продукт.

## 1. Концепция дизайна: "Premium Dark Glass"
Основная идея — использование глубоких темных оттенков, эффектов размытия (glassmorphism) и выверенных акцентов.

### Цветовая палитра (tokens)
- **Backgrounds:** Глубокий сине-черный (#0a0f18) для сайдбара, более светлый графитовый (#111827) для основного контента.
- **Accents:** Основной синий (#3b82f6) с градиентами, мягкие семантические цвета (Success: #10b981, Danger: #ef4444).
- **Glass Effect:** Использование `rgba(255, 255, 255, 0.03)` для карт с `backdrop-filter: blur(12px)`.

## 2. Этапы реализации

### Этап 1: Обновление дизайн-системы (Tokens & Theme)
- [ ] **Рефакторинг `theme.ts`:**
    - Настройка Ant Design ConfigProvider с использованием кастомных токенов.
    - Увеличение `borderRadius` до 12px для всех основных элементов.
    - Настройка теней: использование мягких, глубоких теней вместо резких границ.
- [ ] **Обновление типографики:**
    - Подключение шрифта `Inter` (основной) или `Outfit` (для заголовков).
    - Четкое разделение весов шрифта (Medium 500 для интерфейсного текста).

### Этап 2: Сайдбар и Шапка (Header & Sidebar)
- [ ] **Логотип:** Замена текущего значка грузовика на современный минималистичный SVG-логотип.
- [ ] **Навигация:**
    - Добавление плавных переходов (transition) при наведении.
    - Улучшение активного состояния — мягкая подсветка вместо "жесткого" синего блока.
- [ ] **Поиск:** Улучшение поля поиска в шапке (сделать его более компактным и стильным).

### Этап 3: Таблица и Фильтры (Orders Page)
- [ ] **Секция фильтров:** 
    - Группировка фильтров в аккуратный тулбар.
    - Использование `Input.Search` и `Select` с кастомными стилями (без границ, с фоном).
- [ ] **Таблица заказов:**
    - Увеличение высоты строк для "воздуха".
    - Кастомные бейджи для статусов (Status Tags) с эффектом свечения (glow).
    - Улучшение читаемости колонок "Откуда" и "Куда" с помощью аккуратных иконок.

### Этап 4: Карта и Менеджмент-панель (The Map Card)
- [ ] **Панель управления (Overlay):**
    - Изменение ярких плиток (Активные, Свободные и т.д.) на более благородные оттенки с использованием градиентов.
    - Добавление микро-иконок в каждую плитку.
    - Улучшение стилей переключателей (Switch) и кнопок внизу панели.
- [ ] **Карта:** Стилизация Leaflet под темную тему (использование темных слоев, например, CartoDB Dark Matter).

### Этап 5: Микро-взаимодействия и Полировка
- [ ] **Анимации:**
    - Плавное появление элементов при загрузке страницы (Framer Motion).
    - Hover-эффекты для строк таблицы.
- [ ] **Empty States:** Красивые иллюстрации для пустых состояний (если нет заказов или водителей).
- [ ] **Loading States:** Стилизованные скелетоны (Skeleton Screen) вместо обычных лоадеров.

## 3. Оценка текущего состояния (Review)
- **Сильные стороны:** Структура понятна, основные функциональные блоки на месте, выбранная темная тема актуальна.
- **Что исправить:** 
    - Цвета плиток управления слишком "кричащие".
    - Кнопки пагинации и выбора кол-ва строк выглядят стандартно.
    - Логотип упрощает общий вид приложения.
